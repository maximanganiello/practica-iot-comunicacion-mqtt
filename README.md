# Práctica IoT con comunicación MQTT

Proyecto desarrollado con ESP32 y una interfaz web en HTML, CSS y JavaScript para la adquisición de datos de sensores, el control de actuadores y la comunicación entre dispositivos mediante un broker MQTT.

- `cliente-esp32/cliente-esp32.ino` → Cliente MQTT implementado en ESP32.
- `interfaz-web.html` → Interfaz web para monitoreo, control y chat.
